# Telegram-Movie-Bot
Pull Based Telegram Bot, der mithilfe der [The Movie Database](https://www.themoviedb.org/) API verschiedene Informationen zu Filmen liefert.  

### Commands
* /movie [movie name]
* /poster [movie name]
* /description [movie name]
